form of:
{{documentation subpage}}
This is the most general of the [[:Category:Form-of templates|form-of templates]], which use [[Module:form of/templates]]. It can be used to apply the correct styling of the definition line to any description as the first parameter.

Use this template for forms that don't fit the mold. For instance, the zeroeth-person plural subjunctive of a highly unusual word. If there is already another template available, it's preferred to use that instead. This template '''should not''' be used in other templates, only directly in entries.

==Parameters==
This template takes the same general parameters as {{temp|m}}, and it uses the same post-processing on the parameters. This includes script detection, removing diacritics, processing embedded links, and so on.

; {{docparam|1|'''(required)'''}}
: The [[WT:LANGCODE|language code]] of the term linked to (which this page is a form of). See [[Wiktionary:Languages]]. <small>The parameter {{para|lang}} is a deprecated synonym; please do not use. If this is used, all numbered parameters move down by one.</small>
; {{docparam|2|'''(required)'''}}
: The definition to display, without the "of" at the end.
; {{docparam|3|'''(required)'''}}
: The term to be linked to.
; {{docparam|4}}
: Alternate text to display as the link title, if different from the term.
; {{docparam|t}} or {{docparam|5}}
: A gloss or short translation of the word. <small>The parameter {{para|gloss}} is a deprecated synonym; please do not use.</small>
; {{docparam|tr}}
: Transliteration for non-Latin-script words, if different from the automatically-generated one.
; {{docparam|ts}}
: Transcription for non-Latin-script words whose transliteration is markedly different from the actual pronunciation. Should not be used for IPA pronunciations.
; {{docparam|id}}
: A sense id for the term, which links to anchors on the page set by the {{temp|senseid}} template.
; {{docparam|sc}}
: Script code to use, if script detection does not work.

inflection of:
{{documentation subpage}}
This template is used to create definition lines for inflected (non-lemma) forms of other terms. It use [[Module:form of]] as a back-end.

This template takes the same general parameters as {{temp|l}} and {{temp|m}}, and it uses the same post-processing on the parameters. This includes script detection, removing diacritics, processing embedded links, and so on.

==Parameters==
; {{docparam|1|'''(required)'''}}
: The [[WT:LANGCODE|language code]] of the lemma linked to (which this page is an inflection of). See [[Wiktionary:Languages]]. <small>The parameter {{para|lang}} is a deprecated synonym; please do not use. If this is used, all numbered parameters move down by one.</small>
; {{docparam|2|'''(required)'''}}
: The lemma form of the term that this is an inflection of. This is used to create a link.
; {{docparam|3}}
: The alternative display form of the lemma. This works like the third parameter of {{temp|l}} and {{temp|m}}.
; {{docparam|4|'''(required)'''}}, {{docparam|5}} ... etc.
: One or more grammar tags to show. These give the definition by describing the relevant grammatical properties of this inflected form. A grammar tag can be any text; the tags are shown separated by spaces.
: Certain tags are recognised as shortcuts and are automatically expanded. For example, <code>1</code> is automatically expanded to "first-person", while <code>p</code> becomes "plural". For a full list of recognised shortcuts, see [[Module:form of/data]].
; <code>{{!}};</code>
: Separing two inflections of the same word.
; {{docparam|t}}
: A gloss or short translation of the word. <small>The parameter {{para|gloss}} is a deprecated synonym; please do not use.</small>
; {{docparam|tr}}
: Transliteration for non-Latin-script words, if different from the automatically-generated one.
; {{docparam|ts}}
: Transcription for non-Latin-script words whose transliteration is markedly different from the actual pronunciation. Should not be used for IPA pronunciations.
; {{docparam|id}}
: A sense id for the term, which links to anchors on the page set by the {{temp|senseid}} template.
; {{docparam|sc}}
: Script code to use, if script detection does not work.

==Parameters==
This template takes the same general parameters as {{temp|m}}, and it uses the same post-processing on the parameters. This includes script detection, removing diacritics, processing embedded links, and so on.

; {{docparam|1|'''(required)'''}}
: The [[WT:LANGCODE|language code]] of the primary entry term linked to (which this page is a variant spelling of). See [[Wiktionary:Languages]]. <small>The parameter <code>|lang=</code> is a deprecated synonym; please do not use. If this is used, all numbered parameters move down by one.</small>
; {{docparam|2|'''(required)'''}}
: The type of spelling (e.g. "obsolete", "alternative", "British standard", etc.).
; {{docparam|3|'''(required)'''}}
: The term to be linked to.
; {{docparam|t}}
: A gloss or short translation of the word.
