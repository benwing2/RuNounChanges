alternative plural of:
{{documentation subpage}}
Use this template to show the simple definition for an entry that is the plural inflection of the alternative spelling of a primary entry. (It is preferable to use this template to create a simple alternative-spelling plural article such as [[kilometres]] rather than a redirect: the simple article can later be expanded with the alternative's specifics, such as etymology or quotations.)

==Parameters==
''Positional (unnamed) parameters:''
* {{docparam|1|'''required'''}} the [[WT:LANGCODE|language code]]
* {{docparam|2|'''required'''}} The singular form of the entry for which this is a plural term (i.e., the singular form of the alternative spelling).
* {{docparam|3|'''required'''}} The plural form of which this is itself an alternative spelling.
''Named parameter:''

==Example==
<tt><nowiki># {{alternative plural of|en|kilometre|kilometers}}</nowiki></tt>

form of:
{{documentation subpage}}
This is the most general of the [[:Category:Form-of templates|form-of templates]], which use [[Module:form of/templates]]. It can be used to apply the correct styling of the definition line to any description as the first parameter.

Use this template for forms that don't fit the mold. For instance, the zeroeth-person plural subjunctive of a highly unusual word. If there is already another template available, it's preferred to use that instead. This template '''should not''' be used in other templates, only directly in entries.

==Parameters==
This template takes the same general parameters as {{temp|m}}, and it uses the same post-processing on the parameters. This includes script detection, removing diacritics, processing embedded links, and so on.

; {{docparam|1|''required''}}
: The [[Wiktionary:Languages|language code]] of the desired target link.
; {{docparam|2|''required''}}
: The definition to display, without the "of" at the end.
; {{docparam|3|''required''}}
: The term to be linked to.
; {{docparam|4}}
: Alternate text to display as the link title, if different from the term.
; {{docparam|t}} or {{docparam|5}}
: A gloss or short translation of the word.
; {{docparam|tr}}
: Transliteration for non-Latin-script words, if different from the automatically-generated one.
; {{docparam|ts}}
: Transcription for non-Latin-script words whose transliteration is markedly different from the actual pronunciation. Should not be used for IPA pronunciations.
; {{docparam|id}}
: A sense id for the term, which links to anchors on the page set by the {{temp|senseid}} template.
; {{docparam|sc}}
: Script code to use, if script detection does not work.

inflection of:
{{documentation subpage}}
This template is used to create definition lines for inflected (non-lemma) forms of other terms. It use [[Module:form of]] as a back-end.

This template takes the same general parameters as {{temp|l}} and {{temp|m}}, and it uses the same post-processing on the parameters. This includes script detection, removing diacritics, processing embedded links, and so on.

==Parameters==
; {{docparam|1|''required''}}
: The [[Wiktionary:Languages|language code]] of the term.
; {{docparam|2|''required''}}
: The lemma form of the term that this is a form of. This is used to create a link.
; {{docparam|3}}
: The alternative display form of the lemma. This works like the third parameter of {{temp|l}} and {{temp|m}}.
; {{docparam|4|''required''}}, {{docparam|5}} ... etc.
: One or more grammar tags to show. These give the definition by describing the relevant grammatical properties of this inflected form. A grammar tag can be any text; the tags are shown separated by spaces.
: Certain tags are recognised as shortcuts and are automatically expanded. For example, <code>1</code> is automatically expanded to "first-person", while <code>p</code> becomes "plural". For a full list of recognised shortcuts, see [[Module:form of/data]].
; <code>{{!}};</code>
: Separing two inflections of the same word.
; {{docparam|t}}
: A gloss or short translation of the word.
; {{docparam|tr}}
: Transliteration for non-Latin-script words, if different from the automatically-generated one.
; {{docparam|ts}}
: Transcription for non-Latin-script words whose transliteration is markedly different from the actual pronunciation. Should not be used for IPA pronunciations.
; {{docparam|id}}
: A sense id for the term, which links to anchors on the page set by the {{temp|senseid}} template.
; {{docparam|sc}}
: Script code to use, if script detection does not work.

Use this template to show the simple definition for an entry that is the passive participle inflection of a primary entry. See also [[Template:present passive participle of]] and [[Template:past passive participle of]].

==Usage==
For most entries, use in the definition line:
 <nowiki># {{passive participle of|(term)|lang=xx}}</nowiki>

==Parameters==

; <code>1=</code>
: The lemma to link to. For Arabic, this should include vowel diacritics.
; <code>lang=</code>
: The language code, see [[Wiktionary:Languages]]. Defaults to "en" for English (but encouraged even then).
; <code>2=</code>
: The anchor text to show. Normally unnecessary, and derived from the lemma.
; <code>3=</code> or <code>gloss=</code>
: Gloss (short definition).
; <code>tr=</code>
: Transliteration. Only for non-Latin text and even then often unnecessary, in the case of languages with auto-transliteration.

{{documentation subpage}}
Use this template to show the simple definition for an entry that is the past participle of a primary entry.

'''Special note for English''': This template should only be used when the template {{temp|en-past of}} ("simple past tense and past participle of") cannot be used.

==Usage==
It is used in the definition line:
 <nowiki># {{past participle of|(word)|lang=en}}</nowiki>
E.g., on "[[woven]]":
:<code>===Verb===</code>
:<code># <nowiki>{{</nowiki>past participle of|weave|lang=en}}</code>
:<h3>Verb</h3>
:# {{past participle of|weave|lang=en}}

Note that users can customize how the output of this template displays by modifying their monobook.css files. See [[:Category:Form-of templates|“Form of” templates]] for details.

==Parameters==
This template is not language specific, and can be used for all languages. The <code>lang=</code> parameter sets the language of the word in question, and applies the appropriate language and script codes. It also adds the entry to the categories of that language, and creates a link to that language's section on the target page.

: <code>lang</code> = used to specify the language code (i.e. ''en'' for English).

An example:

For the ''English'' word  ''[[woven]]'' (past participle of ''[[weave]]''):

:<code><nowiki>{{past participle of|weave|lang=en}}</nowiki></code>

This would place the article ''[[woven]]'' in [[:Category:English past participles]]

Use this template to show the simple definition for an entry that is the reflexive form of a primary entry.

For most reflexive entries, use in the definition line:
 <nowiki># {{reflexive of|lemma}}
# {{reflexive of|lemma|alternative display form|lang=language code}}</nowiki>

==Parameters==
The first parameter is required.  It is the lemma (usually the plain infinitive) to which to link.

The second parameter is optional.  It is the alternate display form of the lemma, often used to link to a language-specific section of the target entry.  For example, on {{m|es|lavarse}}:
 <nowiki>===Verb===
{{head|es|verb forms}}

# {{reflexive of|lavar|lang=es}}</nowiki>
:<big>'''Verb'''</big>
:{{head|es|verb forms|head=lavarse}}
:
:# {{reflexive of|lavar|lang=es}}
